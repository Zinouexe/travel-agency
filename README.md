# Atlas Voyage

A modern, premium travel agency operations console built with Spring Boot and Oracle database integration.

## Overview
Atlas Voyage provides an elegant, light-themed web interface for managing your travel agency's core operations from a single cockpit:
- Manage **Customers** and their tailor-made itineraries.
- Explore the **Service Catalog** (accommodations, activities, modes of transport).
- Oversee active **Trips** across the globe.

## Technologies
* **Backend:** Spring Boot, Java
* **Database:** Oracle Database (with Object-Oriented Schema Support)
* **Frontend:** Thymeleaf Templates, Bootstrap 5, Custom CSS

## Quickstart
1. Ensure your Oracle Database connection variables are properly set in `application.properties`.
2. Run the application:
   ```bash
   ./mvnw spring-boot:run
   ```
3. Navigate to `http://localhost:8080/`
